NetMatch* options
 

The main frame of NetMatch* contains three tabbed panels:

  • "Matching" panel (Fig. 1), to specify the target and the query graphs and run the matching task;
  • "Significance" panel (Fig. 2), for the statistical significance of the query as a motif according to a specific random model;
  • "Motif library" panel (Fig. 3), which contains a set of predefined queries for the matching task.
